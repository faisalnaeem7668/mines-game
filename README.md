# Mines Game

A strategic risk-reward game where players reveal tiles to find diamonds while avoiding hidden mines. Built with React 19 and Tailwind CSS 4.

## How to Play

1. Deposit money - Add demo balance to start playing
2. Choose bet amount - Select your bet amount
3. Pick mines count - Adjust risk level from 1 to 24 mines
4. Click tiles - Reveal diamonds to increase multiplier
5. Cash out anytime - Secure your winnings before hitting a mine
6. Hit a mine - Game ends and bet is lost

## Features

- Deposit system with demo balance
- 5x5 interactive game grid
- Random mine placement algorithm
- Dynamic win multiplier calculation
- Auto-pick random tile option
- Cash out mechanism
- Fully responsive design for mobile and tablet

## Tech Stack

- React 19
- Tailwind CSS 4
- Vite 8
- Custom React hooks for game logic
